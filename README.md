# Welcome to Maganets 🛒

A wishlist feature for e-commerces.

### [🌍 DEMO](https://maganets.netlify.app/)

### 📚 Stack Used:

- React 17.0.2
- npm 8.3.0

### 🏃 Running Steps:

1. `git clone https://github.com/lucasvribeiro/maganets.git`
2. `cd maganets`
3. `npm i` or `npm install`
4. `npm start`
